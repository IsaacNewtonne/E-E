# Changelog

## 1.0.0 - 2026-06-13

- Reworked the primary workflow around nightly multi-game score sheets.
- Added spreadsheet-style singles and doubles row entry with automatic points.
- Added keyboard-first entry, paired doubles creation, and stronger night validation.
- Added player editing, inactive roster support, and CSV exports.
- Redesigned the dashboard as an infographic performance report.
- Added configurable pool formats, race length, points, and ranking rules.
- Added scheduled, postponed, completed, editable, and removable matches.
- Added player profiles, images, streaks, averages, history, and head-to-head records.
- Added season archives and historical standings.
- Added JSON backup import and export with legacy-data migration.
- Added automated league-domain tests and GitHub Actions checks.
- Added native Windows desktop launcher and executable build.
