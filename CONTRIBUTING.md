# Contributing

## Development

1. Run `run.bat` to open the app.
2. Make focused changes using the existing dependency-free HTML, CSS, and JavaScript.
3. Run `npm run check` before submitting a change.

League rules belong in `league-core.js` with tests in `tests/`. Browser rendering and
event handling belong in `app.js`.

## Pull Requests

- Describe the user-facing behavior changed.
- Include tests for league-rule changes.
- Include before/after screenshots for visual changes.
- Keep unrelated refactors out of the same pull request.
