# EvitaativE Pool League Design System

## Direction

EvitaativE should feel like a focused late-night pool hall: dark, precise, competitive,
and easy to scan. The logo is the visual source of truth. Use black surfaces, cool
silver text, and a restrained red accent.

## Tokens

- Background: `#08090b`
- Raised surface: `#111317`
- Elevated surface: `#181b20`
- Border: `#2b3038`
- Primary text: `#f4f5f6`
- Muted text: `#969ca6`
- Accent: `#e50920`
- Accent hover: `#ff263a`
- Positive: `#32c47b`
- Warning: `#f2b84b`

## Typography

- Headings: condensed system sans-serif, uppercase, tight line height.
- Body: clean system sans-serif for fast scanning.
- Scores and statistics: tabular numerals.

## Layout

- Desktop uses a fixed left rail and a fluid content canvas.
- Mobile collapses navigation into a bottom bar.
- Cards use 16px radii, subtle borders, and minimal shadows.
- Spacing follows an 8px base rhythm.
- Dashboard prioritizes visual reporting: KPI tiles, horizontal bars, donut charts,
  pairing meters, and compact performance signals.

## Motion

- Keep transitions under 200ms.
- Use motion only for hover, selection, dialogs, and view changes.
